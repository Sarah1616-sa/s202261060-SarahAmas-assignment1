# Sarah Almas - Personal Portfolio Website

A responsive personal portfolio website built as part of the Software Engineering coursework at KFUPM. This project showcases my skills, projects, and professional background as a Software Engineering student.

## Project Description

This portfolio website serves as a professional online presence, featuring:
- **About Me**: Personal introduction and background
- **Skills**: Technical competencies in programming, UI/UX design, and AI
- **Projects**: Showcase of academic and personal projects
- **Contact**: Form for reaching out (frontend only, no backend)

## Features

- Responsive design that works on desktop, tablet, and mobile devices
- Dark/Light theme toggle with user preference persistence
- Clean, modern UI using CSS Grid and Flexbox
- Interactive contact form with validation
- Smooth navigation with mobile hamburger menu

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom properties, Grid, Flexbox, media queries
- **JavaScript**: Theme toggle, form validation, mobile navigation
- **No external frameworks**: Built with vanilla HTML, CSS, and JavaScript

## Setup Instructions

### Running Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/[your-username]/[repository-name].git
   cd assignment-1
   ```

2. **Open in browser**:
   - Simply open `index.html` in your web browser
   - Or use a local server (recommended):
     ```bash
     # Using Python
     python -m http.server 8000

     # Using Node.js (if you have npx)
     npx serve
     ```

3. **Access the site**:
   - Direct file: Open `index.html` in browser
   - Local server: Navigate to `http://localhost:8000`
